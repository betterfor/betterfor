# 🙋‍♂️ Hello

我是betterfor！目前正在从事云原生相关工作。

先给云原生下个定义：cloud+native，简单说就是把本地应用搬到云上去，复杂说就是建立在云上的多重提高效率的方式方法，摆脱本地环境和实际环境的差别，使所有人各司其职，减少沟通成本，利好社恐 🐶

> Stay hungry，stay foolish.
